yêu cầu: sử dụng hệ điều hành almalinux 8,9 hoặc rocky linux 8,9 hoặc ubuntu
<code>curl -sO https://github.com/wptangtoc/neovim-wptangtoc/blob/main/neovim-wptangtoc.sh && bash neovim-wptangtoc.sh</code>
